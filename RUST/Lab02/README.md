# LAB 2 ASSIGNMENT
In questo repository potete trovare il testo del laboratorio 2 di Programmazione di Sistema A.A. 2026:
- **Lab2.md**: testo in Markdown
- **Lab2.pdf**: testo in PDF

Per compilare ed eseguire su terminale all'interno di questa cartella il comando `cargo run`, mentre per eseguire i test il comando è `cargo test`.

I test sono all'interno di una cartella `tests` che non va modificata in alcun modo, ma solo visionata.

All'interno della cartella sono presenti 15 tests che possono essere usati per verificare la correttezza del vostro programma.

Scrivete il vostro codice all'interno della cartella `src`, partendo dal file `main.rs` come punto di ingresso per il vostro programma.
